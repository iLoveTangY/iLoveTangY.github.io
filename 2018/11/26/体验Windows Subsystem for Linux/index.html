安装配置Windows Subsystem for Linux

发表于 2018-11-26 | 更新于 2018-11-27 | 分类于 生产力

​ 因为最近在看CS:APP这本神书，后续还打算看APUE，而它们都需要Linux编程环境。但是我又不想完全抛弃Windows(毕竟还有游戏要玩^_^)，实验室的机器配置不是很好，虚拟机跑起来太慢，双系统又太麻烦。刚好听说Windows 10有WSL，就安装体验了以下，以下是安装教程，大部分参考这里。

启动Windows Subsystem for Linux

有两种方式启动WSL：

通过命令行：

以管理员身份打开Powershell。执行如下命令：

Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Windows-Subsystem-Linux

安装结束后重启电脑。

通过控制面板

打开控制面板，找到启用或关闭Windows功能，找到并勾选适用于Windows的Linux子系统：

完成后重启电脑。

安装Linux发行版

打开win10应用商店，搜索WSL。目前支持一下这些Linux发行版：

选一个你喜欢的安装就好。

完成后可以在最近添加中找到安装的Linux发行版：

打开之后设置用户名密码即可。可以将其固定到开始面板，方便以后打开。

WSL和Windows主系统之间的文件系统是可以互相访问的。

如果在WSL中访问Windows系统的文件，可在根目录下/mnt/看到对应Windows盘符字母的文件夹，通过这些文件夹即可访问Windows的文件系统。

如果在Windows系统中访问WSL的文件，可在Windows系统中找到已安装Linux发行版的应用数据文件夹，所有Linux系统的数据都在那个文件夹（通过应用商店安装的Linux发行版本质上也是一个应用）。Linux发行版的数据文件夹在C:\Users\{你的用户名}\AppData\Local\Packages\{Linux发行版包名}\LocalState\rootfs路径内，在rootfs文件夹内我们会看到在WSL中的根目录。
